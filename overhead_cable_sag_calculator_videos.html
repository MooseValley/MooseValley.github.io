    Training Video #1: Sag v3.x Installation Walkthrough

    See your Registration Confirmation email for further details, links.

    This video takes you through all of the steps necessary to install Sag Calculator v3.x:

    • ** Step 1 of 4: Download ZIP file. See your Registration Confirmation email for details, links.

    • ** Step 2 of 4: Extract the contents of the ZIP file. Run installer EXE as "administrator"

    • ** Step 3 of 4 (Updated): Windows Vista or later users (Windows 7, 8, 10, etc) need to download Windows Help support (WinHelp.exe or WinHlp32.exe) using either of these download links:

      Download #1:
      https://www.dropbox.com/s/khs0lqz723myj68/winhlp32-windows-7-x86-x64-komeil_upd.zip?dl=0

      Download #2:
      http://www.mediafire.com/file/u0updhss784xgfb/winhlp32-windows-7-x86-x64-komeil_upd.zip

      Unzip the contents of the ZIP file to a temporary directory. e.g. c:\temp\ Right click on "Install.cmd" and select "Run as Administrator". A command window will open and WinHLP will be installed, and you will see a bunch of messages ending with: "The operation completed successfully."

    • ** Step 4 of 4: Run Sag Calculator as "administrator" and enter Registration codes provided.

    Downloading the Training Videos to your Computer:

    There are many ways to download videos from YouTube.

    For example, in FireFox there are add-ons like "ANT Video Downloader", "Easy YouTube Video Downloader", and "Flash Video Downloader", but there are plenty of other add-ons that do this as well. (In FireFox: Tools -> Add-ons and in the search box at the top right, type YouTube download and a bunch of add-ons will be listed, such as those mentioned earlier, etc - install whatever add-on you like and read the instructions for using it). Once installed, restart your web browser. These tools usually add a drop-down box below the video in YouTube and you can use this to select the type of video (high def, low def, etc) you want to download.

    My preferred way to download YouTube videos is to use YouTube-DL (a free, open source, command line tool):
    https://github.com/rg3/youtube-dl

    and you can run commands like this (at the command prompt / terminal prompt) to download my videos: 

    	youtube-dl.exe   https://www.youtube.com/watch?v=RU6KTwB8uk4
	youtube-dl.exe   https://www.youtube.com/watch?v=lqINChWBsSk
	youtube-dl.exe   https://www.youtube.com/watch?v=S8d3mTQmyrI

    For watching the videos on your PC, Mac, iPad, Android, Linux, etc, I highly recommend VideoLan VLC Media Player (freeware):
    http://www.videolan.org/
