Sag Calculator TRIAL version - Limitations and Information:

    Until you Register your usage of (i.e. purchase) Sag Calculator, the program is a "demonstration" or "trial" version which has the following limitations:

    1. The TRIAL version is an OLD version of Sag Calculator - so it will *not* contain all of the latest bug fixes, or the latest functionality.

    2. The user cannot save any data to file or disk.

    3. The user cannot print any report, or copy any report data to the Windows' clipboard.

    4. Every now and again you will be informed of how many times you have run Sag Calculator and asked to consider registering your usage of it.

    5. The TRIAL version cannot be made fully functional. It is impossible to do this because selected functionality has been physically removed from the program - it is not just disabled, it has been phsically removed. The user cannot register the TRIAL version to remove any of the above limitations of the TRIAL version.

    After registering the program (i.e. completing the Registration Form and paying the registration fee), you will receive a FULL version of Sag Calculator which is fully functional and has none of the above limitations.

    SAG Calculator is shareware, and the whole idea of shareware is that you can try the program out and really put it through its paces before you by it.:) So, please download the trial version and have a good play around with it, read the "Program Overview" in the on-line help, work through the "Getting Started - Quick Start Guide" in the on-line help, enter some cable settings, do some Sags for a some cable spans, try out the various Sag methods, and make sure the program does just what you need / want / require. (The trial version can do everything the full version can do except for the above limitations).

    I *highly* recommend that at least one person from your company - e.g. an engineer who will be doing the sagging in the field - downloads the trial version of Sag Calculator and really gives it a good try-out and test to make sure they are happy with the program and that it does what you need *before* you place an order or register the program.


Sag Calculator TRIAL version - Download and Installation Instructions:

    Below are the instalation instructions for the TRIAL Version of Sag Calculator. The installation process for the TRIAL Version of Sag Calculator for computers running Windows involves three (3) steps. Please make sure you follow these steps carefully and exactly. If you make a mistake and do one of the steps incorrectly, just re-do the step again and then continue on with the subsequent steps. e.g. if you make a mistake in Step 2, then just do that step again and move on to the subsequent steps - there's no need to uninstall everything or start from Step 1 again for example.

    • ** Step 1 of 3: Download the TRIAL version of Sag Calculator:
      Download via GitHub: Sag_Calculator__v3.3u3__TRIAL_Installer.zip
      Certified 100% Virus Free (VirusTotal.com)

    • ** Step 2 of 3: Extract the contents of the ZIP file to a temporary directory (such as c:\temp\), and run installer EXE as "administrator".
      NOTE: ZIP support is built into all versions of Windows from Windows XP onwards. If you are using a version of Windows prior to Windows XP, then and use WinZIP or similar software to extract the contents of the ZIP file.

    • ** Step 3 of 3: Windows Vista or later (7, 8, 10, etc) download Windows Help support:
      http://support.microsoft.com/kb/917607
      You will probably need to run Microsoft Genuine Advantage when prompted.

    Registered Users - PLEASE NOTE:

    • Do not use the above installation steps for the FULL / Registered version of Sag Calculator.
    • The installation steps for the FULL / Registered version of Sag Calculator are different to those for the TRIAL version.
    • When you register (i.e. purchase) Sag Calculator, you will receive a Registration Confirmation email which contains special, detailed step-by-step installation instructions that need to be followed for the FULL / Registere version of Sag Calculator.
