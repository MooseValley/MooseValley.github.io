My Old Apple 2 Software: (Freeware)

Below is some of the ancient software that I wrote in the very early 1980's for the Apple 2 Computer. A computer that is still fondly remembered and close to my heart even after all of these years ...

NOTE: I have only provided the non-optimised versions of the AppleSoft BASIC programs below. For example, FOR ... NEXT loops span multiple lines, rather than being optimised for the BASIC interpreter onto 1 line of code, and so on. This is because the non-optimised versions are much easier to read / understand, and I figure that anyone accessing the code now wants to understand it so they can perhaps write their own programs or maybe even enhance / change my programs.

To run the programs listed below on modern computer hardware (Windows, Mac OS/X, Linux, etc), you will need to search for an Apple 2 Emulator and download and install this on your computer. For Windows users, I recommend AppleWin.

Address Book :: 1982 Apple 2 version

Address Book is one of the many, many dozens of programs I wrote way back in 1982 on my Apple 2 computer.

At a little over 380 "40 character" lines of code, this was a fairly *big* program way back in 1982 !

I spent a lot of time writing this program, and had a lot of fun doing it.

This program is *extremely* limited, but someone might find it useful. It is really only here for fun / historical purposes.

Text File Editor :: 1982 Apple 2 version

Text File Editor is another of the many, many dozens of programs I wrote way back in 1982/83 on my Apple 2 computer.

At a little over 460 "40 character" lines of code, this was also a fairly *big* program way back in 1982/83.

Editing and using text files on an Apple 2 is a real pain in the neck - but this program makes it easier.

This program is *extremely* limited - it is basically a real simple line editor for text files on the Apple 2 - but someone might find it useful.

To edit / view text files you would be much better off using something like the great text file editor in Locksmith v6.0 (which didn't come out until 1986 - years after I wrote this program) or a word processor. (such as Zardax, Homeword, Sandy's, or some other word processor) - which were around but I didn't have way back in 1982. (Hence the reason I wrote this program ...)

So, this program is virtually useless today, and is only here for fun / historical purposes.

Words Worth - Crossword and Scrabble Companion :: 1982 Apple 2 version

Long before I wrote the Windows version of my Words Worth - Crossword and Scrabble Companion there existed an Apple 2 version of the program, which I wrote way back in 1982.

The basic list of 938 words (available for use by non-registered users of the Windows Version ) was actually extracted from the original floppy disk of my 1982 Apple 2 version of Words Worth - Crossword and Scrabble Companion. This list of 938 words is downloadable here.

So, this word list data file now spans generations of computers, generations of operating systems, and more than 20 years of time. Pretty cool hey !
